## ページフォルダ内部構造について

| フォルダ名     | 内容                                                  |
| -------------- | ----------------------------------------------------- |
| \_components   | 1 つの uiParts とロジックの組み合わせファイル         |
| \_contexts     | ページ別でデータを管理する際の provider、ロジック関係 |
| アンダーバなし | \_components を組み合わせたコンポーネント             |
